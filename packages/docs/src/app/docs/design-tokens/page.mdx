# Design Tokens

Orbi's design token system is the single source of truth for all visual properties. Every component consumes these tokens — no hardcoded values anywhere.

## Usage

Import the CSS file once at your app root:

```tsx
import "@orbi/tokens/tokens.css";
```

Then reference tokens in your own CSS:

```css
.my-card {
  background: var(--orbi-color-surface);
  padding: var(--orbi-space-4);
  border-radius: var(--orbi-radius-md);
  box-shadow: var(--orbi-shadow-sm);
}
```

### Semantic

| Token | Purpose |
|-------|---------|
| `--orbi-color-background` | Page background |
| `--orbi-color-surface` | Card/section surfaces |
| `--orbi-color-surface-raised` | Elevated surfaces |
| `--orbi-color-text-primary` | Primary text |
| `--orbi-color-text-secondary` | Secondary text |
| `--orbi-color-text-muted` | Muted/hint text |
| `--orbi-color-border` | Default borders |
| `--orbi-color-ring` | Focus ring |
